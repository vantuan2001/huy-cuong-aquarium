## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [https://huy-cuong-aquarium.vercel.app](https://huy-cuong-aquarium.vercel.app) with your browser to see the result.
